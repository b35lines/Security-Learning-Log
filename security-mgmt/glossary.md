# 情報セキュリティマネジメント用語集

試験対策や実務で重要となる用語をカテゴリ別にまとめています。

## 1. 情報セキュリティの3要素（CIA）
- **機密性 (Confidentiality)**: 許可された人だけが情報にアクセスできること。
- **完全性 (Integrity)**: 情報が正確で、改ざんされていないこと。
- **可用性 (Availability)**: 必要な時にいつでも情報が利用できること。

## 2. リスクマネジメント
- **リスクアセスメント**: リスクを特定、分析、評価する一連のプロセス。
- **リスク対応**: 
  - **回避**: リスクの原因を取り除く。
  - **移転 (転嫁)**: 保険への加入など、他者に肩代わりさせる。
  - **低減**: 対策を講じて影響を小さくする。
  - **保有 (受容)**: 対策をとらず、許容範囲内として受け入れる。

## 3. 技術的対策
- **IDS/IPS**: 侵入検知システム / 侵入防止システム。
- **WAF**: Webアプリケーションに特化したファイアウォール。
- **SIEM**: 複数の機器のログを一元管理し、相関分析を行う仕組み。
